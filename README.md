# bash-csv-parser-example

Parsing CSV files with Bash IFS

## How to run this script?

```bash
./parse.sh
```

## Sample output

CSV input:

```
Fruit,Price(USD)
Orange,0.2
Pineapple,0.5
Mango,1.4
```

Output:

```
Orange      : $0.20
Pineapple   : $0.50
Mango       : $1.40
```
